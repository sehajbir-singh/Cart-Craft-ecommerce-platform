# Ecommerce Platform

A full-stack ecommerce web application built with separate **admin**, **frontend**, and **backend** folders.  
The platform allows customers to browse products, manage cart items, place orders, and complete payments, while the admin panel is used to manage products, orders, and users.

## Features

### User Side
- User registration and login
- Browse products
- Product details page
- Add to cart / remove from cart
- Update item quantity
- Checkout flow
- Order placement
- Responsive design for mobile and desktop

### Admin Side
- Admin login
- Add, edit, and delete products
- View and manage orders
- Manage users
- Dashboard overview

### Backend
- REST API architecture
- Authentication and authorization
- Database integration
- Secure route handling
- Order and product management logic
